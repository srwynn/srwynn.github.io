Research

My research aims to reduce uncertainty in the effective radiative forcing from anthropogenic aerosols by examining how aerosol–cloud interactions are represented in global climate models.

  • Base-state dependence of parameter sensitivity for the Twomey effect

    Climate models use uncertain parameters to represent aerosol–cloud processes. To identify which parameters are most important, scientists commonly use a multitude of simulations where these parameters are varied uniformly. We show that restricting our analysis to simulations matching the base cloud state gives a different and more meaningful answer about which parameters drive the Twomey Effect. This study provides a simple model agnostic framework for calculating a constrained global sensitivity analysis, through this we find regime-based dependence of parameter sensitivity for the Twomey effect.
